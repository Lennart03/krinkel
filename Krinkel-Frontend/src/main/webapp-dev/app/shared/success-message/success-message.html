  2. Huwelijken, of andere belangrijke familiale momenten zijn ook geldige redenen om je uit te schrijven. Krinkel tijdelijk verlaten kan niet. Je schrijft je uit voor x augustus.
  3. Overlijdens in de familie
  4. Ziekte of ongeval met doktersattest
  5. Vakantie die plots wordt ingetrokken door de werkgever
